# レビュー感情分析による業務改善支援

## ■ 概要
本プロジェクトでは、ECサイトのレビューから顧客の感情を分析し、
業務改善に活用できる情報を抽出することを目的とする。

## ■ 背景
ECサイトでは大量のレビューが投稿されるが、
人手で全てを確認することは困難である。

そのため、レビューを自動的に分類し、
改善点を可視化する仕組みが求められる。

## ■ 使用技術
- Python
- pandas
- scikit-learn
- seaborn / matplotlib

## ■ 分析内容
1. データ前処理（小文字化・記号除去・ストップワード除去）
2. TF-IDFによる特徴量作成
3. Logistic Regressionによる感情分類
4. モデル評価（F1スコア・混同行列）
5. 単語重要度の可視化

## ■ 結果
- レビューの感情分類が可能となった
- ネガティブ要因（品質・期待不一致）を抽出できた
- ポジティブ要因（味・満足度）を特定できた
